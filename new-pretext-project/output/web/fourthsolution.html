    Section 7 Fourth Solution

    This solution is by induction on the power \(k\) with the multiplication algorithm. This solution will prove that if the base-\(2n\) digits of \(n^{k-1}\) are all at least \(2(d+1)\) then the base-\(2n\) digits of \(n^k\) are all at least \(d+1\text{.}\)
    üîó
    The concept of this solution is simple. Multiply \(n^{k-1} \cdot n\) using the elementary school multiplication algorithmin in base \(2n\text{.}\) If every digit of \(n^{k-1}\) is at least \(2d\text{,}\) then every carry is between \(d\) and \(n-1\text{.}\) Therefore every digit of the product is of the form \(x + r \bmod (2n)\) where \(x\) is either \(0\) or \(n\text{,}\) that is, \(n\) times a digit \(\bmod (2n)\text{,}\) and \(d \le r \le n-1\text{.}\) In particular, every digit is at least \(d\text{.}\) Note that in particular, it is necessary to recognize that each carry is at most \(n-1\text{.}\) If \(n\) carried over to an odd digit of \(n^{k-1}\text{,}\) then the corresponding digit of \(n^{k}\) would be \(0\) which is less than \(d\text{.}\)
    üîó

    Example 7.1. Example multiplication by \(5\) in base \(10\).

    To follow the previous reasoning and the Lemma, it helps to carefully examine a multiplication base \(10\text{.}\) Consider the multiplication algorithm applied to multiplying \(5^3 = 625\) by \(5\text{.}\) There is no initial carry so set \(r_0 = 0\text{.}\) First \(5\) multiplies \(5\) giving \(25\) so the unit digit in the product will be \(\operatorname{rem}(5 \cdot 5 + 0, 10) = 5\) and the first carry will be \(r_1 = \lfloor \frac{5*5 + 0}{10} \rfloor = 2\text{.}\) The next step in the multiplication is \(5\) times \(2\) equals \(10\) plus the carry of \(2\) making \(12\text{.}\) Then the next digit will be \(\operatorname{rem}(12, 10) = 2\) and the next carry will be \(r_2 = \lfloor \frac{5*2 + 2}{10} \rfloor = 1\text{.}\) The next-to-final step in the multiplication is \(5\) times \(6\) equals \(30\) plus the carry of \(1\) making \(31\text{.}\) Then the third-from-right digit will be \(\operatorname{rem}(31, 10) = 1\) and the final carry will be \(r_3 = \lfloor \frac{5*6 + 1}{10} \rfloor = 3\text{.}\) is \(5\) times the phantom fourth digit of \(0\) equals \(0\) plus the carry of \(3\) making \(3\text{.}\) Then the fourth-from-right digit will be \(\operatorname{rem}(3, 10) = 3\) and the final carry will be \(r_4 = \lfloor \frac{5*0 + 3}{10} \rfloor = 0\text{.}\) The multiplication is complete.
    üîó

    Lemma 7.2. Multiplication Algorithm.

    Let the base-\(2n\) digits of \(A\) be \(a_{m}a_{m-1} \dots a_{1}a_{0}\text{.}\) In other words, \(0 \le a_i \lt 2n\) and \(A = \sum_{i=0}^m a_i (2n)^{i}\text{.}\) Iteratively define the digits of \(n \cdot A \) as \(c_i\) with carries \(r_i\) as \(r_0 = 0\text{,}\) \(c_i = \operatorname{rem}(n a_i + r_i, 2n)\text{,}\) and
    \begin{equation*} r_{i+1} = \left\lfloor \frac{n a_i + r_i}{2n} \right\rfloor\text{.} \end{equation*}
    Then
    1. \(0 \le c_i \le 2n\) for each \(i \ge 0\text{.}\)
      üîó
    2. \(0 \le r_i \lt n\) for each \(i \ge 0\text{.}\)
      üîó
    3. \(\displaystyle n \cdot A = \sum_{i=0}^{m+1} c_i (2n)^{i}\)
      üîó
    üîó
    üîó

    Proof.

    1. The first item is true by definition since \(c_i\) is a remainder modulo \(2n\text{.}\)
      üîó
    2. Prove the second item by induction on \(i\text{.}\) The base case is true since \(r_0=0\) and then if \(r_i \lt n\)
      \begin{equation*} 0 \le r_{i+1} \le \left\lfloor \frac{n(2n-1) + (n-1)}{2n} \right\rfloor = n-1\text{,} \end{equation*}
      establishing the induction step.
      üîó
    3. To prove the third item, let \(s_k = \sum_{i=0}^{k} c_i (2n)^i\text{.}\) By induction on \(k\text{,}\) \(s_k \equiv nA \mod (2n)^{k+1}\text{.}\) Since \(n \cdot A \lt n \cdot (2n)^{m}\) and \(0 \le s_{m+1} \lt (2n)^{m+1}\text{,}\) this implies \(n \cdot A = s_{m+1}\text{,}\) completing the proof.
      üîó
    üîó

    üîó
    The base case for the induction is \(k=1\) is true if \(N = d\text{.}\) That is, for an odd integer \(n > N\) the base-\(2n\) representation of \(n\) which will be the single ‚Äúdigit‚Äù \(n\) which then satisifies \(n \gt N = d\text{.}\)
    üîó
    Apply the lemma to \(n^{k-1}\) with digits \(a_{m}a_{m-1}\dots a_{1}a_{0}\text{.}\) Sppose that \(a_i \ge 2(d+1)\) for all \(i\text{.}\) Multiplying \(n^{k-1} \cdot n = n^{k}\) with digits \(c_i\) and carries \(r_i\text{.}\) Then \(c_0 = n \gt d\) since \(a_0 = n \gt d\text{.}\) On the other hand, if \(i \ge 1 \text{,}\) then
    \begin{equation*} r_i = \left\lfloor \frac{n a_{i-1} + r_{i-1}}{2n} \right\rfloor \gt \left\lfloor \frac{n \cdot 2(d+1) + 0}{2n} \right\rfloor = 1 \end{equation*}
    so all carries are at least \(d+1\text{.}\) But \(\operatorname{rem}(n \cdot a_{i}, 2n)\) is either \(0\) or \(n\text{,}\) so
    \begin{equation*} 0 + (d+1) \le \operatorname{rem}(n \cdot a_{i}, 2n) \le n + (n-1) \end{equation*}
    which means that \(\operatorname{rem}(n \cdot a_{i}, 2n) + r_i\) is between \(d+1\) and \(2n-1\) so
    \begin{equation*} c_i = \operatorname{rem}( \operatorname{rem}(n \cdot a_{i}, 2n) + r_{i}, 2n) \gt d \text{.} \end{equation*}
    üîó
    As an alternative argument, if the base-\(2n\) digits of \(n^{k-1}\) are \(a_{m}a_{m-1}\dots a_{1}a_{0}\) and the digits of \(n^{k-1} \cdot n = n^{k}\) are \(c_{k-1}c_{k-2}\dots c_{2}c_{1}c_{0}\text{,}\) then multiplying \(n^{k}\) by \(2\) gives the base-\(2n\) number \(a_{m}a_{m-1}\dots a_{1}a_{0}0\text{.}\) If one of th digits \(c_i \le d\text{,}\) then when multiplying it by \(2\text{,}\) the carry is at most \(1\text{,}\) so \(a_{i} \le 2 c_i + 1 \le 2d + 1\) which contradicts \(a_{i} \ge 2(d+1)\text{.}\)
